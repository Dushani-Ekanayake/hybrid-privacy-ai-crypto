How It Works

1
User inputs data

Income, credit score, debt, and employment years are entered locally in the browser.

2
Homomorphic Encryption

Features are encrypted using CKKS scheme (TenSEAL). The server receives only ciphertext — it can't read the actual values.

3
Cloud inference on ciphertext

The AI model performs a dot product directly on the encrypted vector using plaintext weights.

4
Differential Privacy

Before returning the result, Laplace noise (ε=1.0) is added. This prevents inference attacks from the output.

5
Result returned

The noisy probability is returned. The user decrypts and sees their result. Server learns nothing.
